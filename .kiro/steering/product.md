# Product Vision: SidePilot

## Overview
**SidePilot** is a Chrome extension that enables AI-powered browser automation using **any LLM provider** the user chooses—not just Claude.

## Tagline
> "Your AI Co-Pilot in the Browser"

## Target Users
- Developers who want browser automation with their preferred LLM
- Users with existing API keys for various providers
- Teams requiring on-prem/self-hosted LLM options (Ollama)
- Power users wanting fine-grained control over AI models

## Core Value Proposition
**"Bring Your Own LLM"** - Use any of 40+ LLM providers (Anthropic, OpenAI, Google, DeepSeek, Ollama, etc.) for browser automation. Unlike Claude's extension which only works with Claude, SidePilot is provider-agnostic.

## Key Features

### Must Have (MVP)
1. Multi-provider support (40+ LLMs like Cline)
2. Full browser automation (click, type, navigate, screenshot)
3. Side panel chat interface
4. Permission system for safety
5. Saved shortcuts/prompts

### Should Have
6. Workflow recording (step-by-step capture)
7. Tab group management
8. Network/console tracking
9. Notifications

### Innovation (Differentiators)
10. Model capability warnings (vision, tools, streaming)
11. MCP Connector - expose tools to external LLMs (Cline, Aider)

## Success Metrics
- All 13 browser tools working
- 10+ providers tested
- Sub-second response latency
- Zero permission bypasses

## Non-Goals
- GIF recording
- Voice narration for recording
- Mobile browser support
